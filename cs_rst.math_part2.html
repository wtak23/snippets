4. Advanced Latex Mathematics on Sphinx

Bunch of stuffs from https://en.wikibooks.org/wiki/LaTeX/Advanced_Mathematics

i wanted to see what renders correctly.

http://meta.math.stackexchange.com/questions/5020/mathjax-basic-tutorial-and-quick-reference

Note

Stuffs that didn’t render will appear red

4.1. Some self-notes I created

4.1.1. bold for lower-case greek

  • use \boldsymbol to bold lower greek letters (\mathbf doesn’t do it):

    \mathbf{\beta}\\
\boldsymbol{\beta}
\[\begin{split}\mathbf{\beta}\\ \boldsymbol{\beta}\end{split}\]

4.1.2. Defining own function

I got this on top or the rst file. Then i can use them anywhere in the current rst file (cross-file support seems non-existent).

.. math::

    \DeclareMathOperator*{\argmax}{arg\,max}
    \DeclareMathOperator*{\argmin}{arg\,min}
    \newcommand{\mybeta}{\boldsymbol{\beta}}
    \newcommand{\bx}{\boldsymbol{x}}
    \newcommand{\bb}[2]{\argmax_{#1}\ell(#1,#2)}
\[\DeclareMathOperator*{\argmax}{arg\,max} \DeclareMathOperator*{\argmin}{arg\,min} \newcommand{\mybeta}{\boldsymbol{\beta}} \newcommand{\bx}{\boldsymbol{x}} \newcommand{\bb}[2]{\argmax_{#1}\ell(#1,#2)}\]

Default alignment: flush right

\argmin_\mybeta \frac{1}{n}\sum_{i=1}^n \ell(\bx_i,\mybeta)
    + \lambda \Psi(\mybeta)
\\
\bb{\mybeta}{\bx}
\[\begin{split}\argmin_\mybeta \frac{1}{n}\sum_{i=1}^n \ell(\bx_i,\mybeta) + \lambda \Psi(\mybeta) \\ \bb{\mybeta}{\bx}\end{split}\]

Flush left: Use \beglin{align} env to align to right (centering? not sure how at the moment)

\begin{align}
    &\argmin_\mybeta \frac{1}{n}\sum_{i=1}^n \ell(\bx_i,\mybeta)
        + \lambda \Psi(\mybeta)
    \\
    &\bb{\mybeta}{\bx}
\end{align}
\[\begin{split}\begin{align} &\argmin_\mybeta \frac{1}{n}\sum_{i=1}^n \ell(\bx_i,\mybeta) + \lambda \Psi(\mybeta) \\ &\bb{\mybeta}{\bx} \end{align}\end{split}\]

Note

I ignored all the ones pertaining to equation labeling and referencing. Those are understandably limited in sphinx and mathjax.

Warning

It appears that latex \begin{env} blocks needs to be in their own .. math block to be rendered properly.

Example: below won’t render

.. math::

    \begin{equation}
     f(x)=(x+a)(x+b)
    \end{equation}

    \begin{equation}
     g(x)=(x)(b)
    \end{equation}
\[\begin{equation} f(x)=(x+a)(x+b) \end{equation}\]\[\begin{equation} g(x)=(x)(b) \end{equation}\]

If each \begin environment receives its own math directive, it works.

.. math::

    \begin{equation}
     f(x)=(x+a)(x+b)
    \end{equation}

.. math::

    \begin{equation}
     g(x)=(x)(b)
    \end{equation}
\[\begin{equation} f(x)=(x+a)(x+b) \end{equation}\]
\[\begin{equation} g(x)=(x)(b) \end{equation}\]
